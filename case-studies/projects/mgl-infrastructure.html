Project Case Study

Major Gaming League

Complete infrastructure rebuild of a CODM BR ranking server — from 100 members to 3,000+ with 172 channels, 91 roles, and premium forum organization

Consultant Server Admin Data Analyst Sept 2023 - Present

Context & Challenge

In September 2023, I was brought on as Consultant, Server Admin, and Data Analyst for Major Gaming League, a Call of Duty Mobile Battle Royale ranking server. The challenge was significant:

  • Server in baby phase: ~100 members with messy, disorganized structure
  • Complete rebuild required: Existing channels and roles were unusable
  • Competitive ranking focus: Need to support tiered competitive structure for CODM BR
  • Scalability requirement: Infrastructure must support exponential growth

My Approach: I made the decision to clear everything — deleted all existing channels and roles to start with a clean foundation. This allowed me to architect a professional, scalable infrastructure from the ground up.

Key Achievement: Transformed MGL from a disorganized 100-member server into a 3,024-member competitive CODM BR league with enterprise-grade organization, achieving 2,924% growth over 2+ years.

Infrastructure Implementation

1. Complete Server Rebuild

Rather than incrementally patching the existing structure, I executed a full rebuild strategy:

Rebuild Process:

  • Phase 1: Cleared all channels, categories, and roles
  • Phase 2: Designed information architecture for competitive ranking server
  • Phase 3: Implemented 172 channels across logical categories
  • Phase 4: Created 91-role hierarchy supporting staff, ranking tiers, and community
  • Phase 5: Deployed bot ecosystem and automation systems

Result: A professional Discord infrastructure capable of supporting thousands of competitive players with clear navigation, automated workflows, and scalable organization.

5. Forum Channels with Tag System (PREMIUM)

Implemented Discord's forum channels with a sophisticated tagging system for organized team roster management:

Forum channel showing tier-based filtering (TIER 1, TIER 2, TIER 3) for team roster posts

Forum Channel Features:

  • Tag-Based Organization: TIER 1, TIER 2, TIER 3 tags for competitive tier filtering
  • Structured Posts: "Legionaries" team with roster change updates
  • Search & Filter: "Sort & view" controls for easy navigation
  • Post Templates: Standardized format for roster updates (Player out: IGN-LGN, Call. Player in: IGN-SpiriTaLGN, Pro Name...)
  • Post Guidelines: Right panel with "CREATING A POST" instructions including roster change format, player limitations, EU/NA player rules

Forum Guidelines Enforced:

  • Player Limitation: Ensure roster doesn't exceed maximum of 6 players
  • EU/NA Limitation: Max 2 EU/NA players per roster, only 1 allowed to play in a lobby
  • Tag Selection: Choose appropriate tier tag based on team's competitive level
  • Title Format: Post title should include team name for easy identification

Impact: Forum channels with tags created a self-organizing system for team roster management, reducing admin workload and making roster changes transparent and searchable.

  • ● Verified Female — Gender diversity recognition (113 members)

  • ● Music Bots — Audio bots (4 members)

Role Distribution Analysis:

Of 3,026 total members (as shown in screenshots), the role breakdown demonstrates:

  • Competitive Focus: 733 members in ranked tiers (Tier 1-4) + 151 in prestige tiers = 29% active competitive players
  • Small Admin Team: ~20 staff/admin roles maintaining order for 3,000+ members
  • Community Base: 2,944 "Community" role members = general participant pool
  • Specialized Functions: Dedicated roles for GFX, Data Analysis, Casting, and FPR create professional operational structure

Design Rationale: The 91-role system supports operational needs (moderation, event management, fair play), competitive structure (4 ranked tiers + 4 prestige tiers), and community engagement (verified female, regional roles). Color-coding and hierarchy provide immediate visual clarity for members.

8. Bot Ecosystem & Automation

Deployed a multi-bot ecosystem to automate server operations and community engagement:

Core Bot Stack:

  • Sapphire — Primary utility bot for server management and moderation
  • Koya — Community engagement and leveling system
  • Astro — Additional moderation and automation features
  • Jockie Music Bot (All Variants) — Voice channel music and entertainment
    • Multiple instances for high-quality audio across voice channels
  • Ticket Bot — Support ticket system for member assistance and dispute resolution
  • ProBot ✨ — Premium features for advanced automation

Automation Highlights:

  • Onboarding Flow: Automated role assignment via customization questions (region, game mode, content preferences)
  • Ranking Management: Tier role assignment based on competitive performance
  • Moderation: Auto-mod rules, anti-spam, raid protection via Sapphire and Astro
  • Support System: Ticket Bot handles player disputes, roster changes, and compliance issues
  • Engagement: Koya provides leveling and activity tracking to encourage participation
